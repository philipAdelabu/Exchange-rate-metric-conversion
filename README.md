 # An Exchange Rate and Metric Conversion web application
##  A Wdd 330 final project 

It is a minimalistic project requirement for WDD330 course from the Bragham Young University (BYU). The WDD330 is also a requirement for the Software Development  Associat Degree programme.

### Description
An application that convert currency values across the world. It also has the capability to conver dimension metric from one dimension to another.

### API needed 
- (Exchange Rate API) [https://exchangerate-api.com]
- (Unit converted API) [https://unitconvr.com]
